Основные компоненты CSS фреймворков обычно включают в себя:

1. Сетка (Grid System): Сетка позволяет разделять контент на колонки, что облегчает создание адаптивного дизайна для различных устройств и размеров экранов.


2. Типографика (Typography): Это включает в себя стили для заголовков, параграфов, списков и других текстовых элементов.


3. Формы (Forms): Стилизация форм, таких как текстовые поля, чекбоксы, радиокнопки и кнопки.


4. Кнопки (Buttons): Стилизация стандартных и пользовательских кнопок.


5. Навигация (Navigation): Создание навигационных меню, выпадающих меню, хлебных крошек и т.д.


6. Карточки (Cards): Стилизация карточек, которые используются для представления информации в удобном виде.


7. Иконки (Icons): Наборы иконок для использования в веб-приложениях и сайтах.


8. Модули для всплывающих окон (Modal Popup): Компоненты, которые обеспечивают отображение всплывающих окон или модальных окон.


9. Компоненты форматирования контента (Content Formatting Components): Это может включать таблицы, аккордеоны, вкладки и другие компоненты, которые форматируют и отображают контент на странице.


10. Анимации (Animations): Библиотеки анимаций для создания интерактивных элементов и переходов.


Это только основная часть компонентов, которые обычно включаются в CSS-фреймворки. Каждый фреймворк может также предлагать свои дополнительные компоненты, в зависимости от его особенностей.
